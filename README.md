# LeapMotion Gesture Project


### Required Dependencies
 * GTK+
 * Leap Motion Driver
 * Leap Motion SDK

### Optional Dependencies
If not installed some functions couldn't work
 * bspwm
 * xdotool

### Get Started

```sh
  $ git clone https://github.com/joaquingx/LeapMotion/
  $ cd LeapMotion
  $ cmake .
  $ make
  $ ./leapObject
```
